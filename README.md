# app-registry 

app.json registry webservice, backed by a markdown file in a GitHub wiki.

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install app-registry --save
```


## Tests

```sh
npm install
npm test
```
